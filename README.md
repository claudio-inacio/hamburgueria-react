🍔 BURGER HERO — Sistema de Pedidos para Hamburgueria

O Burger Hero é uma aplicação web desenvolvida com React, criada com o objetivo de simular o fluxo real de pedidos em uma hamburgueria.
O usuário pode navegar pelos produtos, adicionar ao carrinho, remover itens, alterar quantidades e finalizar a compra utilizando cartão ou PIX.

Além disso, há suporte para adicionar novos produtos, permitindo que o projeto simule funcionalidades reais de um painel administrativo.

🖼️ Prévia do Projeto


🧠 Sobre o Projeto

O Burger Hero foi desenvolvido como um estudo de componentização, gerenciamento de estado e lógica de carrinho, simulando o fluxo real de uma hamburgueria digital.
A aplicação oferece uma interface simples e eficiente, tornando a experiência intuitiva tanto para o cliente quanto para o administrador.
